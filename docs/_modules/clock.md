Provide 4 (currently, more is planned) trigger outputs synchronized to a main clock.  

- BPM based clock (1 - 300 BPM)
- Frequency based clock (0.01 - 20Hz)
- Per output features:
  - gate time
  - divisor and multiplier
  - output voltage
